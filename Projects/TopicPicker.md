# Project 3: TopicPicker! (Ongoing)
## Description
This is a little Java project I'm working on to develop my Java skills. It uses a hashmap full of subjects (as the keys) and lists of topics (used as the values).

### Features
- Expandable with the ability to add to the subjects easily
- Runs on nearly anything with java
- Simple to use, just using the console
