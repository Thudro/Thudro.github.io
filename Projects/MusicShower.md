# Project 2: MusicShower
## Description
Displays the current music session details from a hosting machine.

### Features
- Uses a host and a client 
- The host uses the windows sdk (winsdk) to get the title and artist from the machine
- The host then hosts that (using flask) as json and is acessable through a get request.
- The client then gets that json from the host.
- The client retrieves the album cover from Itunes
- Then all that information is displayed using a simple flask server and a bit of HTML!
